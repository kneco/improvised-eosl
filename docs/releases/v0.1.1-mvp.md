# v0.1.1-mvp

This is the first downloadable Windows package for the experimental
`window.showModalDialog()` compatibility MVP.

## Download and start

1. Download `ImprovisedEosl-0.1.1-mvp-win-x64.zip` from the release assets.
2. Extract the entire ZIP to a writable folder.
3. Double-click `ImprovisedEosl.Spike.SyncModal.exe`.
4. On the local test page, click `Open child dialog`.

Do not run the executable directly inside the ZIP. The adjacent `config` and `pages`
folders are required.

## Requirements

- 64-bit Windows 10 or Windows 11
- Microsoft Edge WebView2 Runtime

The .NET 8 runtime is included in the ZIP.

## Validation

- the published self-contained executable completed all three `--auto` synchronous dialog cycles and exited with code 0
- all 42 policy tests passed
- package contents include the WebView2 native loader, HTML pages, compatibility configuration, and end-user instructions

## Important

This remains an experimental technical preview for isolated evaluation. It is not Internet
Explorer, a production compatibility guarantee, or a supported business-continuity product.
See `docs/mvp-readiness.md` and `docs/risks-and-limitations.md` before testing a legacy site.
